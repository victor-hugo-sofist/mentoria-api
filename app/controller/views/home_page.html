PRODUTOS FINANCEIROS - API

Esta API é capaz de salvar informações relativas a produtos financeiros e atualizar seus respectivos preços.

Métodos

/--(GET)-- Home page.

/stocks/ --(GET)-- Exibe todos os produtos financeiros cadastrados.

/stock/ITIU4/ --(GET)-- Exibe a informação relativa ao primeiro produto cadastrado com este símbolo.

/new/stock/ --(POST)-- Cadastra um novo produto. É necessário enviar via body, em formato json, as informações "Name", "Symbol" e "Price".
EXEMPLO:
 {
  "Name": "ITI BANCO DIGITAL",
  "Symbol": "ITIU4",
  "Price": 55
 }

/stock/ITIU4/change/price/ --(PATCH)-- Altera o preço do primeiro produto cadastrado com este símbolo. É necessário enviar via body, em formato json, a informação de "Price".
EXEMPLO:
 {
  "Price": 2
 }

/stock/delete/ITIU4/ --(DEL)-- Deleta o primeiro produto com este símbolo cadastrado.

/download/ --(GET)-- Exibe as requisições do postman em formato json. Para incluir as requisições no Postman basta copiar as informações retornadas e importar como 'raw text'.
